mapGuardedSignal()method
public function mapGuardedSignal(signal:ISignal, commandClass:Class, guards:*, oneshot:Boolean = false):void

Map a Command to an instance of a Signal, with Guards

The signal - an instance of ISignal

The commandClass must implement an execute() method

The guards must be a Class which implements an approve() method

or an Array of Classes which implements an approve() method

Parameters

signal:ISignal — The signal instance to trigger this command. Values dispatched by this signal are available for injection into guards and the command.
 
commandClass:Class — The Class to instantiate - must have an execute() method
 
guards:* — The Classes of the guard or guards to instantiate - must have an approve() method
 
oneshot:Boolean (default = false) — Unmap the Class after execution?

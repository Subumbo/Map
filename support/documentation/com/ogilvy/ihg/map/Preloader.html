Classpublic class Preloader
InheritancePreloader MicrositePreloader flash.display.Sprite

Preloader hooks up the view asset with the preloader controller (MicrositePreloader.as which is the base class). Designer can design the preloader_asset.swf in the .fla (./support/Preloader_Asset/Preloader_Asset.fla). Once design is finished the swf needs to be compiled into the preloader via the flex compiler.
